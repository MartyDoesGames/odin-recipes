# odin-recipes

This is my first HTML project. The goal of this project is to create a very basic recipe website. It will consist of an index page with links to a few recipes. It will not look pretty.

This project allows me to practice and demonstrate:
    - Writing HTML boilerplate.
    - Using common HTML tags.
    - Inserting images into HTML pages.
    - Creating lists in HTML.
    - Linking to other HTML pages.
